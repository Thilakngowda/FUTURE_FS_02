# 📊 Client Lead Management System (Mini CRM)

## Overview
The **Client Lead Management System** is a simple Customer Relationship Management (CRM) web application designed to help businesses organize and track potential clients (leads).  

This system allows users to add, manage, and monitor leads efficiently while keeping track of their status and follow-up information. The project demonstrates core concepts of web development and basic business workflow management.

---

## ✨ Features
- Add and manage client leads
- Track lead status *(New, Contacted, Converted)*
- Store lead information such as name, email, source, and notes
- Search and filter leads
- Delete or update lead records
- Simple dashboard showing total leads

---

## 🛠 Technologies Used
- **HTML5** – Structure of the application  
- **CSS3** – User interface and layout styling  
- **JavaScript** – Application logic and interaction  
- **Local Storage** – Client-side data storage
